# 📌 ECG Image Classification using CNN and Optimization Algorithms

## 🚀 Project Overview
This project focuses on classifying ECG (Electrocardiogram) images using Deep Learning. A Convolutional Neural Network (CNN) is implemented and enhanced using optimization algorithms like Harris Hawks Optimization (HHO) and Super Eagle Optimizer.

---

## 🧠 Models Used
- Baseline CNN  
- CNN + Harris Hawks Optimization (HHO)  
- CNN + Super Eagle Optimizer  

---
